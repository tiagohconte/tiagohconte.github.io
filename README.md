# Personal Portfolio

A minimalist personal portfolio website built with pure HTML and CSS.

## Features

- **Typewriter Animation**: Animated text that alternates between "Tiago Conte" and "Software Engineer" with a blinking cursor
- **Social Links**: Direct links to GitHub and LinkedIn profiles
- **Responsive Design**: Optimized for desktop and mobile devices
- **Pure CSS**: No JavaScript or frameworks required

## Technologies

- HTML5
- CSS3
- Google Fonts (Ubuntu Mono)

## Usage

Simply open `index.html` in any modern web browser. No build process or dependencies needed.

## Customization

To personalize the website:

1. **Edit text content** in `index.html`:
   - Update the typewriter text in the CSS keyframes in `styles.css`
   - Change social media links

2. **Modify colors** in `styles.css`:
   - Background: `#2c2c2e`
   - Text: `#ffffff`
   - Icons: `#9ca3af` (hover: `#e5e7eb`)

3. **Adjust animation speed**: Change the `animation` duration in `.typewriter::before` (default: 16s)
